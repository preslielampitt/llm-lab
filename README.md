# cmc-cs040-preslielampitt
 
![doctest](https://github.com/preslielampitt/llm-lab/actions/workflows/doctests.yml/badge.svg)
![integration-test](https://github.com/preslielampitt/llm-lab/actions/workflows/integration-tests.yml/badge.svg)
 
A command-line chatbot powered by Groq's LLM API, built as part of CMC's CSCI040 course. This lab sets up the foundation for the upcoming LLM project.
 
## This lab demonstrates:
1. How to write "test cases for your test cases"
2. How to get LLMs working in GitHub Actions
3. How to let other people `pip install` your projects
 
## Installation
 
```bash
pip install cmc-cs040-preslie
```
 
## Usage
 
```bash
chat
```
 
Once running, type your messages and press Enter. Use `Ctrl+C` to exit.
